Learning

Books:

  • Computer Networking: A Top-Down Approach: An easy to read book that gives an overview on each layer of the TCP/IP model without going too deep into technical specs.

Lectures/Video Courses:

  • CS50x: This is a great introductory course into programming and CS in general as it doesn't only focus on writing code but how computers interact with and run that code.
  • Node.js: I tried a few Node.js courses before descovering this one; unlike others, the instructor goes through almost every line but without overexplaining.
